天気予報アプリ

現在地や任意の都市名を入力して天気情報を取得・表示する天気予報アプリを制作しました。API連携を通して、リアルタイムの天気情報を取得できる構成にしています。

URL

非公開（個人開発のためローカル環境で実行）

使用技術

React / Next.js / TypeScript / Tailwind CSS / Weather API

担当工程

設計、コーディング、API連携、スタイリング

目的・特徴

ReactやNext.jsのフレームワークを活用し、非同期通信・API連携・ルーティングなどのモダンなフロントエンド技術の理解を深めることを目的に制作しました。TypeScriptによる型安全な実装や、Tailwind CSSを用いた効率的なUI構築にも取り組んでいます。

工夫した点

初心者でも使いやすいUIを目指し、気温・湿度・天気アイコンを視覚的に整理。エラー時にはアラート表示でユーザーにフィードバックを返す設計にしています。
